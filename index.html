Welcome to the Easy HTML5 Chat app!

Thank you for visiting this site. This application is a GPL lisenced website featuring a chat client which is run off of websockets and React.js.

Motivation

There are many reasons why I chose this framework to base the app off of:

  1. The site should be easy to use and navigate
    • React is a capable software which is the framework for sites with ease of use such as Facebook, Instagram, and even a web-based application like Wix.
    • Websockets are javascript native. They should be well understood by anyone familiar with websocket programming experience, thus making the code easier to navigate.
  2. Chat should be visible in near real-time
    • Even on a lightweight server like the Raspberry Pi 3, websockets are still an extremely fast, reliable way to pass data back and forth.
  3. The server instance needs to be lightweight (the websocket server is hosted on a Raspberry Pi 3 model B which slated to host multiple projects).
    • Once again, websockets are lightweight and fast enough to be hosted on a Raspberry Pi 3 with reasonable performance.
  4. The user interface should be feature rich.
    • Even large, data driven applications like Wix are easily visualized using React.js.
